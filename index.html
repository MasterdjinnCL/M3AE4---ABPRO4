Elementos del Box Model

Cada elemento HTML es una caja rectangular que consta de margin, border, padding y content. En la siguiente demostración puedes ver todas las partes al mismo tiempo:

Content - El contenido real del elemento
Padding - Espacio interno entre content y border
Border - Borde alrededor del padding
Margin - Espacio externo alrededor del element
